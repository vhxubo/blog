# 发布一个 npm 包 [#18](https://github.com/vhxubo/blog/issues/18)

-  [注册账号](https://www.npmjs.com/) 

- 发布
```
npm adduser
npm publish
```
- 更新版本（也可以直接改动 version）
```
# 升级补丁版本号
$ npm version patch

# 升级小版本号
$ npm version minor

# 升级大版本号
$ npm version major
```

P.S 记得发布，发布之后可能需要**两分钟**才能看到
