## 可以看到 Vue 完整源码

> 项目在 build 时,webpack 中会有一个叫 productionSourceMap 的配置属性,设置成 fasle,项目在打包后就看不到源码目录结构

```
module.exports = {
  productionSourceMap: false
};

```
